## はじめに

このドキュメントは、Markdownの記法を学び、テストするために作成されました。Markdownはシンプルで直感的なマークアップ言語であり、ウェブや文書作成の場面で広く利用されています。以下の内容で、Markdownの各種記法を実演し、それらがどのようにレンダリングされるかを確認していきます。

## セクション1: 基本的な記法

ここではMarkdownの基本的な要素から始めていきます。これらはほとんどのMarkdownエディタやプラットフォームでサポートされています。

### 見出しについて

Markdownでは、異なるレベルの見出しを使用して文書を構造化します。見出しは通常、`#` 記号を使って示されます。

#### 見出しについて　その2

ある日、ジョンという名の青年が、小さな村で目を覚ましました。この村は彼の故郷であり、彼はこの穏やかで美しい場所で育ちました。村の周りには緑豊かな森林が広がっており、小鳥のさえずりや川のせせらぎが心地よいBGMとなっています。ジョンは毎朝、村を囲む道を走ることを日課にしていました。彼は自然の中で過ごす時間を何よりも大切にしており、この地のすべてを愛していました。

```markdown
# レベル1見出し

## レベル2見出し

### レベル3見出し
```

### リストの作成

リストはMarkdownで非常によく使用される機能です。項目の前に`-`または`*`を置くことで、箇条書きリストを作成できます。また、番号を使用することで番号付きリストを作成することも可能です。

#### 箇条書きの例

- ライオン
- トラ
  - ベンガルトラ
  - シベリアトラ
- ヒョウ

#### 番号付きリストの例

1. 秋の味覚
   1. さつまいも
   2. 栗
2. 冬の味覚
   1. 大根
   2. ほうれん草

### テキストの強調

テキストの強調は、文章の要点を伝えるために非常に効果的です。Markdownでは以下の方法で強調を表現します。

- _イタリック_: 一重のアスタリスクまたはアンダースコアで囲む。
- **ボールド**: 二重のアスタリスクまたはアンダースコアで囲む。
- **_イタリックとボールド_**: 三重のアスタリスクで囲む。

### 引用について

引用は情報源を明示するか、重要な情報を強調したいときに使用します。以下に使用例を示します。

## セクション2: メディアとコード

このセクションでは、Markdownを使用してどのようにメディアを表示し、コードをきれいにフォーマットできるかを示します。

### 画像とリンク

Markdownを使用して簡単にリンクや画像を挿入できます。

- リンク例: [Wikipedia](https://www.wikipedia.org)

### コードブロック

コードをより読みやすく表示するために、バッククォート（````）を使用してコードブロックを作成します。

```javascript
// JavaScriptのサンプルコード
function sayHello(name) {
  console.log(`Hello, ${name}!`);
}

sayHello('World');
```

### 表の利用

データを整理して表示するために表を使います。

## セクション3: より高度な使い方

### チェックリスト

タスク管理などに便利なチェックリストをMarkdownで作成することができます。

- [x] 準備をする
- [ ] 実行する
- [ ] フィードバックをもらう

### インラインHTML

Markdownは主にテキストベースですが、インラインでHTMLを使用することで、さらに多機能なコンテンツを作成できます。

### 水平線

Markdownでは、複数の`-`や`*`を使って水平線を引くことができます。これは文書のセクションを分ける際に便利です。
