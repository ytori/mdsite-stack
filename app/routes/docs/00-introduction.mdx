## これは何か

ドキュメント作成にフォーカスした[React Router v7](https://reactrouter.com/)のテンプレートプロジェクトです。

**M**ark**D**own + Static **Site** Generation = **MDSite** Stack

## 特徴

- Markdownからドキュメントサイトを簡単に作成できる
- 最低限でシンプルで理解しやすく、カスタマイズしやすい
- React Router v7の 新機能のprerenderにより、イニシャルロードが早い

## アーキテクチャ

- [React Router v7](https://reactrouter.com/)
- [MDX](https://mdxjs.com/)
- [Tailwind CSS](https://tailwindcss.com/) & [Tailwind Typography](https://github.com/tailwindlabs/tailwindcss-typography)
- [shadcn/ui](https://ui.shadcn.com/)
