---
title: Markdown
description: An explanation of Markdown syntax
---

Markdown is a lightweight markup language for formatting text easily. Here, we introduce some basic Markdown syntax.

## Headings

Headings are created using `#`. The more `#` symbols there are, the lower the heading level.

Example:

```md
### Heading 1

Content for Heading 1

#### Heading 2

Content for Heading 2
```

Rendered as:

### Heading 1

Content for Heading 1

#### Heading 2

Content for Heading 2

## Bold and Italic

- **Bold**: Add `**` or `__` before and after the text.
- _Italic_: Add `*` or `_` before and after the text.

Example:

```md
**This is bold**
_This is italic_
```

Rendered as:

**This is bold**  
_This is italic_

## Lists

### Unordered List

Unordered lists can be created using `-`, `+`, or `*`.

Example:

```md
- Item 1
- Item 2
- Item 3
```

Rendered as:

- Item 1
- Item 2
- Item 3

### Ordered List

Ordered lists are created using numbers followed by a period.

Example:

```md
1. Item 1
2. Item 2
3. Item 3
```

Rendered as:

1. Item 1
2. Item 2
3. Item 3

## Links

Links are created using the format `[Link Text](URL)`.

Example:

```md
[Google](https://www.google.com)
```

Rendered as:

[Google](https://www.google.com)

## Images

Images are created using the format `![Alt Text](Image URL)`.

Example:

```md
![Sample Image](https://via.placeholder.com/150)
```

Rendered as:

![Sample Image](https://via.placeholder.com/150)

## Blockquotes

Blockquotes are created using `>`.

Example:

```md
> This is a blockquote.
```

Rendered as:

> This is a blockquote.

## Code

### Inline Code

Inline code is wrapped in backticks `` ` ``.

Example:

```md
`code`
```

Rendered as:

`code`

### Multiline Code

Multiline code is wrapped in three backticks.

Example:

````md
```ts
const message = 'Hello World!';

console.log(message);
```
````

Rendered as:

```ts
const message = 'Hello World!';

console.log(message);
```

## Horizontal Rules

Horizontal rules are created using `---`, `***`, or `___`.

Example:

```md
---
```

Rendered as:

---

## Tables

Tables are created using `|` and `-`.

Example:

```md
| Header 1 | Header 2 |
| -------- | -------- |
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```

Rendered as:

| Header 1 | Header 2 |
| -------- | -------- |
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
