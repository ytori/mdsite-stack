マークダウン（Markdown）は、テキストを簡単にフォーマットするための軽量マークアップ言語です。ここでは、いくつかの基本的なマークダウン記法を紹介します。

## 見出し

見出しは `#` を使って作成します。`#` の数が多いほど、見出しのレベルが下がります。

例:

```md
### 見出し1

見出し1のコンテンツ

#### 見出し2

見出し2のコンテンツ
```

表示結果:

### 見出し1

見出し1のコンテンツ

#### 見出し2

見出し2のコンテンツ

## 太字と斜体

- **太字**: 文字の前後に `**` または `__` を追加します。
- _斜体_: 文字の前後に `*` または `_` を追加します。

例:

```md
**これは太字です**
_これは斜体です_
```

表示結果:

**これは太字です**
_これは斜体です_

## リスト

### 順不同リスト

順不同リストは、`-`、`+`、または `*` を使って作成できます。

例:

```md
- アイテム 1
- アイテム 2
- アイテム 3
```

表示結果:

- アイテム 1
- アイテム 2
- アイテム 3

### 番号付きリスト

番号付きリストは、数字とピリオドを使って作成します。

例:

```md
1. アイテム 1
2. アイテム 2
3. アイテム 3
```

表示結果:

1. アイテム 1
2. アイテム 2
3. アイテム 3

## リンク

リンクは `[リンクテキスト](URL)` の形式で作成します。

例:

```md
[Google](https://www.google.com)
```

表示結果:

[Google](https://www.google.com)

## 画像

画像は `![画像の説明](画像URL)` の形式で作成します。

例:

```md
![サンプル画像](https://via.placeholder.com/150)
```

表示結果:

![サンプル画像](https://via.placeholder.com/150)

## 引用

引用は `>` を使って作成します。

例:

```md
> これは引用です。
```

表示結果:

> これは引用です。

## コード

### インラインコード

インラインコードはバッククォート `` ` `` で囲みます。

例:

```md
`コード`
```

表示結果:

`コード`

### 複数行のコード

複数行のコードは、3つのバッククォートで囲みます。

例:

````md
```ts
const message = 'Hello World!';

console.log(message);
```
````

表示結果:

```ts
const message = 'Hello World!';

console.log(message);
```

## 水平線

水平線は `---`、`***`、または `___` を使って作成します。

例:

```md
---
```

表示結果:

---

## 表

表は `|` と `-` を使って作成します。

例:

```md
| 見出し 1 | 見出し 2 |
| -------- | -------- |
| セル 1   | セル 2   |
| セル 3   | セル 4   |
```

表示結果:

| 見出し 1 | 見出し 2 |
| -------- | -------- |
| セル 1   | セル 2   |
| セル 3   | セル 4   |
